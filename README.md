# Walmart Sales Dashboard (2010–2012)

Interactive Tableau dashboard analyzing Walmart sales trends, top-performing stores, holiday impacts, and external factors such as temperature and fuel prices.

## Dashboard Preview

![Dashboard](dashboard.png)

## Key Insights

- Total sales reached $6.7B.
- Store 20 generated the highest sales ($301.4M).
- Peak monthly sales reached $288.8M.
- Lowest monthly sales were $163.7M.
- Non-holiday weeks contributed the majority of sales.
- Temperature and fuel prices showed weak relationships with weekly sales.

## Tools Used

- Tableau Public
- SQL
- CSV
- Data Visualization
